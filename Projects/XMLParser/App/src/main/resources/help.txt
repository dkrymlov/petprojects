                                    Привіт!
Щоб почати роботу, відкрий файл з базою студентів,
що живуть в гуртожитку натиснувши Файл -> Відкрити.
На екрані з'явиться таблиця, та шлях до відкритого файлу.
-----------------------------------------------------------------
Щоб виконати пошук по файлу, введіть ключове слово в
поле для введення та нажміть кнопку Пошук. Всі студенти
що відповідають вашому запиту з'являться на екрані.
-----------------------------------------------------------------
Щоб обрати інший спосіб пошуку даних, натисніть на чекбокс DOM/SAX
DOM - галочка не вибрана, SAX - галочка на місці.
-----------------------------------------------------------------
Щоб зберегти відфільтровану інформацію, натисніть
Файл -> Зберегти.
-----------------------------------------------------------------
