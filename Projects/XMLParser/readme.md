# XML Database Parser
# About 

Java program to parse XML database.

# Functions

Import XML database
		
Parse XML database(SAX, DOM)

Show XML database in table format

Find values in database by keywords

Export filtered table in HTML table format(XLST)

# Technologies

UI(Swing+AWT)

Java

Apache Maven

Parsers(SAX Api, DOM Api)

XSLT

JUnit
